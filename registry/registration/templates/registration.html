Registration to Soteria is available to everyone with a ORCID iD!

Following the steps below will grant you the 'Affiliate' status for 30 days after which you can seek sponsorship by a Verified Researcher or become a Verified Researcher yourself. More information on the various roles in Soteria can be found on the About Page.

{{ card_header_check("1. Create an OSG Hub Account") }}

Follow the link to OSG Hub and create an account using the same identity provider that you used to login with Soteria. If you would like to change identity providers you can logout and come back to this page to restart the registration workflow.

Check For OSG Hub Account
{{ card_header_check("2. Link ORCID iD with COmanage") }}
If do not have a ORCID iD

Follow the link to register at ORCID.org and then proceed with the step below.

If you have a ORCID iD

Follow the link to COmanage and link your ORCID iD to your account. Then return and click the 'Check for ORCID iD Link' button below.

Check for ORCID iD Link
{{ card_header("3. Explore Soteria!") }}

If you have completed the previous two steps you are now a registered Affiliate with Soteria. You can explore what this means on the About Page or you can provision your first private project on the Project Page.
